# 📊 Customer-Churn-Analysis-for-a-Telecom-Company-Databel-

## 🔎 Description
Subscription-based companies depend heavily on customer retention.
This project analyzes customer churn for a fictitious telecom company, Databel, using Microsoft Excel.

The goal was to:

* Calculate overall churn rate
* Identify why customers are leaving
* Detect high-risk customer segments
* Provide business recommendations to reduce churn
---
## 🛠 Tools Used
Microsoft Excel:
- Calculated columns
- PivotTables
- PivotCharts
- Slicers
- Dashboard design
---

## 📌 Key KPIs

Total Customers: 6,687
Customers Churned: 1,796
Churn Rate: 26.86%

The churn rate of 26.86% indicates a significant retention issue requiring targeted intervention.

## 📊 Key Insights
1️⃣ Competitive Pressure Drives Churn

The majority of churn is linked to competitor offers, particularly:

* Better devices
* More attractive pricing
* Higher data allowances

This suggests Databel faces aggressive market competition.

2️⃣ Older Customers Are High-Risk

Customers aged 60+ show elevated churn rates.

This indicates:

* Possible price sensitivity
* Poor perceived value
* Need for targeted senior plans

3️⃣ Contract Structure Impacts Retention

Month-to-month customers churn significantly more than long-term contract users.

Long-term contracts appear to reduce churn probability.

4️⃣ Mid-Tier Data Users Are Most Likely to Leave

Customers using between 5–10GB show higher churn than heavy users.

This suggests pricing/value misalignment for mid-tier customers.

## 🧠 Business Recommendations

* Introduce device upgrade incentives
* Launch senior loyalty discount plan
* Encourage migration to annual contracts
* Re-evaluate mid-tier pricing structure
